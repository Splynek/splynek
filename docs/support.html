Support

Splynek is built and maintained by one person. Every email gets a human reply — usually within 48 hours.

Email

info@splynek.app

Best for: licence issues, refund requests, crash reports, or bug reports that reference sensitive URLs or download contents.

Response time: within 48 hours on weekdays; up to 5 days on weekends.

GitHub Issues

github.com/Splynek/splynek/issues

Best for: public bug reports, feature requests, reproductions you're happy to share publicly.

Free-tier source is MIT — pull requests welcome on the issue tracker.

Before you write

A few things that shortcut most support tickets:

  • Update to the latest version. Mac App Store users update automatically. DMG users: GitHub Releases.
  • Check the Gatekeeper signature panel (Splynek → Settings → Security & privacy) if the app won't launch. It tells you exactly which signature check failed.
  • Run the Benchmark tab if downloads are slower than expected. Measures single-interface vs multi-interface throughput — if multi-interface isn't higher, there's a config issue the numbers surface.
  • Reset state by removing ~/Library/Application Support/Splynek/ (DMG) or ~/Library/Containers/app.splynek.Splynek/ (Mac App Store). Splynek restarts fresh.

Refunds

App Store purchases: use Apple's built-in refund flow — open App Store → your profile → Purchased → Report a Problem next to Splynek. Apple usually approves refunds within 24 hours. We don't process App Store refunds ourselves; Apple does.

DMG purchases: there aren't any. The DMG build is free and open-source. If you paid for Splynek elsewhere (a reseller site that shouldn't exist), email us — it's not legitimate.
