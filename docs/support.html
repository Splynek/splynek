Support

Splynek is built and maintained by one person. Every email gets a human reply.

Email

support@splynek.app

Best for: licence issues, refund requests, crash reports, bug reports that mention sensitive URLs or download contents.

Response time: within 48 hours on weekdays; up to 5 days on weekends.

GitHub Issues

github.com/Splynek/splynek/issues

Best for: public bug reports, feature requests, reproductions you're happy to share.

Free-tier source is MIT-licensed and public — pull requests welcome on the issue tracker.

Before you write

A few things to check that will shortcut most support tickets:

  • Update to the latest version. App Store users get updates automatically; DMG users: github.com/Splynek/splynek/releases.
  • Check the Gatekeeper signature panel (Splynek → Settings → Security & privacy) if the app won't launch. It tells you exactly which signature check failed.
  • Run the bench (Benchmark tab) if downloads are slower than expected. Splynek measures single-interface vs multi-interface throughput — if the multi-interface result isn't higher, there's a configuration issue the numbers will surface.
  • Reset state by removing ~/Library/Application Support/Splynek/ (DMG) or ~/Library/Containers/app.splynek.Splynek/Data/ (App Store). Splynek restarts fresh.

Refunds

App Store purchases: use Apple's built-in refund flow — open the App Store app → your profile → Purchased → Report a Problem next to Splynek. We don't process App Store refunds ourselves; Apple does.

DMG purchases: there aren't any. The DMG build is free and open-source. If you paid for Splynek elsewhere (e.g., a reseller site that shouldn't exist), email us — it's not legitimate.
