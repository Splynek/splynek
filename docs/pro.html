Questions people actually ask

Does Splynek Pro phone home?

No. All AI features run locally via Ollama or LM Studio on your machine. The only network calls are (1) the downloads you explicitly request and (2) LAN Bonjour + peer fetches. No telemetry, no analytics, no crash reports. See the privacy policy.

Do I need Ollama? I'm not comfortable with the terminal.

v0.50+ auto-detects LM Studio too — a GUI-first Mac-native alternative that doesn't need a terminal. Either provider works; Splynek picks whichever is running. If neither is installed, the Concierge tab shows two install buttons.

Is this a subscription?

No. Non-consumable Mac App Store In-App Purchase. One-time $29, tied to your Apple ID, usable on every Mac signed into it. Stops working never.

What if I want a refund?

Open App Store → your profile → Purchased → Report a Problem next to Splynek. Apple handles refunds usually within 24 h. We don't see your purchase data, so we can't do it for you.

Do the old HMAC keys from v0.33–v0.43 still work?

No. Pro is StoreKit-enforced from v0.44 forward. If you had an early-tester HMAC key and end up buying the MAS version, email paulo@splynek.app and we'll waive the $29.

Family Sharing?

Not in v1. One key per Apple ID. Will revisit if asked.
